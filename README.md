# Expense_Tracker#
Expense Tracker (Трекер расходов)
Автор: [уратоа ДенисЯзык: Python 3.10

Описание программы
Expense Tracker — это простое графическое приложение для учёта личных финансов. Программа позволяет записывать ежедневные расходы, разделяя их по категориям (например, еда, транспорт, развлечения), и отслеживать общую сумму трат.

Основные возможности:

Добавление расходов с указанием суммы, категории и даты.
Фильтрация списка расходов по категории и/или конкретной дате.
Автоматический подсчёт общей суммы всех расходов.
Сохранение данных в файл expenses.json и их автоматическая загрузка при запуске.
Инструкция по запуску
Установите Python 3.10, если он ещё не установлен на вашем компьютере.
Скачайте или склонируйте этот репозиторий на свой компьютер.
Откройте терминал (командную строку) в папке с проектом.
Запустите приложение, выполнив команду:
bash
Копировать
python expense_tracker.py
Используйте графический интерфейс для работы с программой.
Примеры использования
1. Добавление нового расхода
В поле «Сумма» введите число (например, 500 или 1200.50).
В поле «Категория» введите тип расхода (например, еда, такси, кино).
В поле «Дата» введите дату в формате ДД.ММ.ГГГГ (например, 29.04.2026).
Нажмите кнопку «Добавить расход».Запись появится в таблице ниже.
2. Фильтрация по категории
Введите ключевое слово в поле «Категория» (например, транс).
Нажмите кнопку «Применить фильтр».В таблице отобразятся только те расходы, в названии категории которых встречается введённое слово (например, «транспорт»).
3. Фильтрация по дате
Введите конкретную дату в поле «Дата» (например, 01.05.2026).
Нажмите кнопку «Применить фильтр».В таблице останутся только расходы, совершенные в указанный день.
4. Просмотр общей суммы
Текущая сумма всех добавленных расходов отображается в верхней части окна (над таблицей) и обновляется автоматически при добавлении новых записей или применении фильтров.

5. Сохранение данных
При закрытии окна программы все введённые данные автоматически сохраняются в файл expenses.json. При следующем запуске приложения список расходов будет загружен из этого файла автоматически.
